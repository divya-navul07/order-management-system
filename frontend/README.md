# Frontend - Order Management System

React + TypeScript frontend for the order management system with responsive design and real-time updates.

## Setup

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm run preview
```

## Key Features

- **Product Browsing** - Search, filter, and browse products
- **Shopping Cart** - Full cart management with checkout
- **Order Management** - Track orders and view history
- **User Authentication** - Login and registration
- **Dashboard** - Admin dashboard for order management
